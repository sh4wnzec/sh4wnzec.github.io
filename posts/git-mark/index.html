Vue cli 3 项目创建时，Git Bash 箭头选择无效

发表于 January 4, 2022 | 分类于 （76 Words，阅读时间约 0 Min）

问题

在 Git Bash 输入 vue create xxx 创建项目，选择选项时没有光标；

解决方法

  • 找到 Git Bash 的安装目录，前往 etc 目录，找到 bash.bashrc 文件；
  • 在该文件末尾新增一行 ：alias vue='winpty vue.cmd'
  • 重启 Git Bash 即可。
